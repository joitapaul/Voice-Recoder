# 🎙️ Voice Recorder (Python)



---

## 📌 Project Overview

This Voice Recorder allows users to:
- Record audio using the system microphone
- Save the recorded audio as a WAV file
- Play the output audio using any media player

---

## 🛠️ Tech Stack
